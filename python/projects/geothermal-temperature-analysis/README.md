# Geothermal Temperature Analysis

## Project Overview

This project analyzes geothermal temperature data to understand subsurface thermal conditions and geothermal potential.

## Objectives

- Learn geothermal fundamentals
- Process temperature datasets using Python
- Visualize temperature-depth relationships
- Estimate geothermal gradients

## Skills

- Python
- Pandas
- NumPy
- Matplotlib
- Scientific Computing

## Output

- Cleaned dataset
- Temperature-depth plots
- Geothermal gradient estimation
- Final report
